# MaskedPi v0.1.0

初回プロトタイプ版です。実機確認に必要な最小構成をまとめました。

## 主な追加内容
- WPF ベースのローカル実行型マスキング補助ツール
- `rules.json` / `local_dictionary.json` / `test_cases.json` による設定外部化
- 自治体ローカルルール拡張と辞書ベース動的ルール生成
- ルールテスター画面（ヒット詳細表示・コピー機能）
- 業務別プロファイル切替（common / resident / tax / welfare / education）
- GitHub Actions による通常ビルド（手動実行）
- GitHub Actions によるリリースビルド（手動実行）
- コミット済みリリースノートからの自動 Release 作成

## 注意事項
- 本版はプロトタイプ段階です。
- 実運用前に誤検出・過検出の確認を行ってください。
